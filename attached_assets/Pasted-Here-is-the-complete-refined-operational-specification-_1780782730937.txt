## 1. System Feature Specifications

Your updated dashboard application must feature five fundamental architectural components:

### A. Core Wallet Directory & Bulk Data Importer

* **Bulk Import Processing:** A secure plaintext area that accepts bulk data arrays containing unstructured mnemonic seed phrases, recovery private keys, or raw text blocks. The parser dynamically extracts valid seed components, converts them to standard formats, derivations, and structures them into localized data schemas.
* **Master Ledger Table UI:** High-density, real-time reactive table mapping columns for:
* Public Derivative Address (EVM hex format vs. Native `me1` prefix format mappings)
* Account Verification Tag Status (`Verified` vs `Unverified` network states)
* Real-time Multi-Tier Balance breakdowns



### B. Multi-Tier Real-Time Balance Tracker

* Direct live connection to your specified node endpoint or block explorer index arrays to display three decoupled asset types for every item inside your fleet:
1. **ME-Hub Balance:** Liquid spendable on-chain layer base balance.
2. **Rolling Up Balance:** Dynamic layer execution reward allocations awaiting withdrawal calls.
3. **My Staking Rewards:** Accumulating reward dividends attached to active network validation agents.



### C. Manual Peer-to-Peer Transfer Terminal

* Interactive dashboard screen panel permitting safe, manual transfers of arbitrary amounts of native tokens from any single internal address inside your data array directly to any external recipient address or another internal node, with custom transaction nonce handling to bypass pending bottlenecks.

### D. Multi-Mode Auto-Sweeper Controller

* **Automated Mode Targets:** A selection interface giving you total granular control over which assets to target during automation routines:
* **Mode A (All-Inclusive Sweep):** Sequentially clears out staking claims, extracts rolling pools, balances them inside the local address, and cascades the consolidated payload down the chain.
* **Mode B (ME-Hub Liquid Target Only):** Skips smart contract claim actions entirely. Merely grabs existing liquid ME-Hub balances and sends them down the line.
* **Mode C (Rolling Up Isolation Only):** Only interacts with the Rollup contract layer to claim and clear that balance component.
* **Mode D (Staking Rewards Isolation Only):** Only triggers validation engine contract withdrawal actions.


* **Gas-Reserve Maintenance Safeguard:** The automated logic guarantees that regardless of the target mode chosen, the local wallet must keep a fixed, configurable minimum threshold of base token liquid capital sitting safely inside the address's `ME-Hub` balance. This prevents out-of-gas call drop rejections when invoking contract calls or executing transfer steps.

### E. Classified Backup & Bulk Export Engine

* Single-click extraction feature that compiles your entire workspace. It automatically separates wallets into download packages based on active categorization rules, formatting each row with its full set of generation metadata:
* CSV/JSON File Alpha: **Verified Accounts Archive** (Addresses, derived keys, and passphrases that have passed identity rules)
* CSV/JSON File Beta: **Unverified Accounts Pool** (Raw generated keys awaiting initial deployment, validation, or funding states)

## 3. Production Backend Implementation Engine (`chain_sweeper.py`)

This full-scale code module provides the complete programmatic infrastructure for the dashboard. It features raw string parsing for bulk additions, multi-mode transaction building, internal manual transfers, and cascading auto-sweeping loops that honor your gas reserve requirements.

```python
import time
import os
import re
from web3 import Web3
from eth_account import Account
from mnemonic import Mnemonic

# Core Meta Earth Network System Layer Allocations
HARDCODED_MASTER_ADDRESS = "me1h4fc80gz38ms8tejlj37rxmf7uh6xe25fk0tfx"
STAKING_CONTRACT_HEX = "0x0000000000000000000000000000000000001001"
ROLLUP_CONTRACT_HEX = "0x0000000000000000000000000000000000001002"
SYSTEM_CHECKIN_HEX = "0x0000000000000000000000000000000000001003"

MINIMAL_ABI = [
    {"constant": True, "inputs": [{"name": "_owner", "type": "address"}], "name": "balanceOf", "outputs": [{"name": "balance", "type": "uint256"}], "type": "function"},
    {"constant": False, "inputs": [{"name": "_amount", "type": "uint256"}], "name": "unstake", "outputs": [], "type": "function"},
    {"constant": False, "inputs": [], "name": "claimRollupBalance", "outputs": [], "type": "function"},
    {"constant": False, "inputs": [], "name": "dailyCheckIn", "outputs": [{"name": "success", "type": "bool"}], "type": "function"}
]

class MetaEarthDashboardEngine:
    def __init__(self, rpc_endpoint="https://rpc.me-network.io"):
        self.web3 = Web3(Web3.HTTPProvider(rpc_endpoint))
        self.master_target = HARDCODED_MASTER_ADDRESS
        
        # Instantiate instances for Core Contracts
        self.staking_contract = self.web3.eth.contract(address=Web3.to_checksum_address(STAKING_CONTRACT_HEX), abi=MINIMAL_ABI)
        self.rollup_contract = self.web3.eth.contract(address=Web3.to_checksum_address(ROLLUP_CONTRACT_HEX), abi=MINIMAL_ABI)
        self.checkin_contract = self.web3.eth.contract(address=Web3.to_checksum_address(SYSTEM_CHECKIN_HEX), abi=MINIMAL_ABI)

    def parse_bulk_imports(self, raw_input_text):
        """
        Parses a block of text to extract 12/24-word mnemonics or 64-character hexadecimal private keys.
        """
        found_keys = []
        found_mnemonics = []
        
        # Extract Private Keys via Regex
        hex_matches = re.findall(r'(?:0x)?[a-fA-F0-9]{64}', raw_input_text)
        for match in hex_matches:
            clean_key = match if match.startswith("0x") else "0x" + match
            if clean_key not in found_keys:
                found_keys.append(clean_key)
                
        # Extract Mnemonics lines
        lines = raw_input_text.split('\n')
        mnemo_validator = Mnemonic("english")
        for line in lines:
            clean_line = line.strip().lower()
            word_count = len(clean_line.split())
            if word_count in [12, 15, 18, 21, 24]:
                if mnemo_validator.check(clean_line):
                    if clean_line not in found_mnemonics:
                        found_mnemonics.append(clean_line)
                        
        print(f"📥 Bulk Parse Complete: Identified {len(found_keys)} keys, {len(found_mnemonics)} mnemonics.")
        return {"private_keys": found_keys, "mnemonics": found_mnemonics}

    def fetch_comprehensive_balances(self, query_address):
        """
        Retrieves real-time metric readings from the network across all three balance profiles.
        """
        try:
            me_hub = self.web3.eth.get_balance(query_address)
            staking = self.staking_contract.functions.balanceOf(query_address).call()
            rollup = self.rollup_contract.functions.balanceOf(query_address).call()
            return {"me_hub": me_hub, "staking": staking, "rollup": rollup}
        except Exception as e:
            print(f"⚠️ Balances lookup error for address {query_address}: {e}")
            return {"me_hub": 0, "staking": 0, "rollup": 0}

    def execute_manual_p2p_transfer(self, source_private_key, recipient_address, amount_in_ether):
        """
        Executes a direct manual on-chain transfer from a designated wallet to any destination address.
        """
        try:
            sender_account = Account.from_key(source_private_key)
            sender_addr = sender_account.address
            
            # Format destination inputs properly
            target_recipient = recipient_address
            if recipient_address.startswith("me1"):
                target_recipient = Web3.to_checksum_address("0x" + recipient_address[3:])
                
            nonce = self.web3.eth.get_transaction_count(sender_addr)
            tx = {
                'nonce': nonce,
                'to': target_recipient,
                'value': self.web3.to_wei(amount_in_ether, 'ether'),
                'gas': 21000,
                'gasPrice': self.web3.eth.gas_price,
                'chainId': self.web3.eth.chain_id
            }
            
            signed_tx = self.web3.eth.account.sign_transaction(tx, source_private_key)
            tx_hash = self.web3.eth.send_raw_transaction(signed_tx.rawTransaction)
            print(f"💸 Manual P2P Transfer Processed! Tx Hash: {tx_hash.hex()}")
            return tx_hash.hex()
        except Exception as e:
            print(f"❌ Manual transfer routing execution rejected: {e}")
            return None

    def dispatch_bulk_daily_checkin(self, targeted_private_keys):
        """
        Sequentially executes check-in contract interactions across selected keys.
        """
        print(f"📅 Initiating checking-in tasks for {len(targeted_private_keys)} wallets...")
        for idx, p_key in enumerate(targeted_private_keys):
            try:
                addr = Account.from_key(p_key).address
                nonce = self.web3.eth.get_transaction_count(addr)
                
                tx = self.checkin_contract.functions.dailyCheckIn().build_transaction({
                    'from': addr, 'nonce': nonce, 'gas': 70000, 'gasPrice': self.web3.eth.gas_price, 'chainId': self.web3.eth.chain_id
                })
                signed = self.web3.eth.account.sign_transaction(tx, p_key)
                tx_hash = self.web3.eth.send_raw_transaction(signed.rawTransaction)
                print(f"   [{idx+1}] Check-In confirmed for: {addr} | Tx: {tx_hash.hex()}")
                time.sleep(1)
            except Exception as e:
                print(f"   [{idx+1}] Skipping check-in for key index due to fee error: {e}")

    def execute_advanced_cascading_sweep(self, structured_keys_list, target_sweep_mode="all", gas_reserve_ether=0.01):
        """
        Runs the cascading extraction chain across the wallet fleet.
        Supported modes: 'all', 'me_hub_only', 'rollup_only', 'staking_only'
        """
        total_wallets = len(structured_keys_list)
        reserve_wei = self.web3.to_wei(gas_reserve_ether, 'ether')
        
        print(f"🔄 Launching Sweeper. Mode: [{target_sweep_mode.upper()}]. Reserve Margin: {gas_reserve_ether} MEC")

        for index, current_key in enumerate(structured_keys_list):
            try:
                current_acc = Account.from_key(current_key)
                current_addr = current_acc.address
                
                # Assign output target endpoint link
                if index < total_wallets - 1:
                    dest = Account.from_key(structured_keys_list[index + 1]).address
                    is_terminal_master = False
                else:
                    dest = self.master_target
                    is_terminal_master = True

                print(f"\n[Wallet {index+1}/{total_wallets}] Address: {current_addr} ➔ Dest: {dest}")
                
                balances = self.fetch_comprehensive_balances(current_addr)
                nonce = self.web3.eth.get_transaction_count(current_addr)
                gas_price = self.web3.eth.gas_price

                # Condition 1: Claim Staking Rewards if valid
                if target_sweep_mode in ["all", "staking_only"] and balances["staking"] > 0:
                    if balances["me_hub"] > (80000 * gas_price):
                        print(f"   🔓 Pulling Staking Rewards: {self.web3.from_wei(balances['staking'], 'ether')} MEC")
                        tx = self.staking_contract.functions.unstake(balances["staking"]).build_transaction({
                            'from': current_addr, 'nonce': nonce, 'gas': 80000, 'gasPrice': gas_price, 'chainId': self.web3.eth.chain_id
                        })
                        signed = self.web3.eth.account.sign_transaction(tx, current_key)
                        self.web3.eth.send_raw_transaction(signed.rawTransaction)
                        nonce += 1
                        time.sleep(2)

                # Condition 2: Claim Rollup Allocations if valid
                if target_sweep_mode in ["all", "rollup_only"] and balances["rollup"] > 0:
                    if balances["me_hub"] > (80000 * gas_price):
                        print(f"   🌀 Extracting Rollup Rewards: {self.web3.from_wei(balances['rollup'], 'ether')} MEC")
                        tx = self.rollup_contract.functions.claimRollupBalance().build_transaction({
                            'from': current_addr, 'nonce': nonce, 'gas': 80000, 'gasPrice': gas_price, 'chainId': self.web3.eth.chain_id
                        })
                        signed = self.web3.eth.account.sign_transaction(tx, current_key)
                        self.web3.eth.send_raw_transaction(signed.rawTransaction)
                        nonce += 1
                        time.sleep(2)

                # Condition 3: Final Base Layer Forwarding Step
                updated_balances = self.fetch_comprehensive_balances(current_addr)
                base_liquid_pool = updated_balances["me_hub"]
                transfer_fee = 21000 * gas_price
                
                # Establish maximum available allocation allowed to leave the wallet link
                if is_terminal_master:
                    # Final node clears everything out down to zero gas fees
                    max_sweepable = base_liquid_pool - transfer_fee
                else:
                    # Chained nodes retain your gas reserve to pay for future processing tasks
                    max_sweepable = base_liquid_pool - reserve_wei - transfer_fee

                if max_sweepable > 0:
                    recipient_payload = dest
                    if is_terminal_master and dest.startswith("me1"):
                        recipient_payload = Web3.to_checksum_address("0x" + dest[3:])
                        
                    sweep_tx = {
                        'nonce': nonce,
                        'to': recipient_payload,
                        'value': max_sweepable,
                        'gas': 21000,
                        'gasPrice': gas_price,
                        'chainId': self.web3.eth.chain_id
                    }
                    
                    signed_sweep = self.web3.eth.account.sign_transaction(sweep_tx, current_key)
                    tx_hash = self.web3.eth.send_raw_transaction(signed_sweep.rawTransaction)
                    
                    if is_terminal_master:
                        print(f"🏁 MASTER TERMINAL MET: Consolidated funds landed at final address! Tx: {tx_hash.hex()}")
                    else:
                        print(f"   💸 Forwarded {self.web3.from_wei(max_sweepable, 'ether')} MEC. Awaiting receipt...")
                        self.web3.eth.wait_for_transaction_receipt(tx_hash)
                else:
                    print("   ⚠️ Forwarding skipped: Local liquid balance is below the gas reserve margin threshold.")

            except Exception as e:
                print(f"   ❌ Chain pipeline execution failure at item index {index}: {e}")
                break

```
